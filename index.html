A propos de moi

Je suis actuellement en formation pour devenir développeur web fullstack,
en ce momment je suis entrain de chercher une entreprise qui pourrais m'accepter en tant que alternant.

J'ai entamé mon parcours par un CAP vente au lycée Georges Brassens à Villepinte, suivi d'un BAC professionnel dans le même établissement. Ces premières étapes m'ont permis de développer aptitudes commerciales et en communication. J'ai ensuite entrepris une remise à niveau en français, en mathématiques et en bureautique afin d'améliorer mes compétences. Cette formation a été une étape cruciale, me permettant de renforcer mes compétences et de découvrir de nouvelles perspectives. Ma véritable révélation est survenue lors de ma participation à la Plateforme Métiers du Tertiaire au centre Belle Alliance à Groslay. C'est dans ce cadre dynamique que j'ai découvert ma vocation pour le développement web. En explorant les différentes facettes de ce domaine, j'ai réalisé que créer des applications web intuitives et performantes était ma véritable passion. Aujourd'hui, je souhaite me spécialiser dans le développement web. J'ai pour désir de concevoir des applications qui offrent une expérience utilisateur optimale, en utilisant les technologies les plus récentes et innovantes.

Mes compétences

HTML

HTML est un langage de balisage utilisé pour structurer et présenter le contenu des pages web.

CSS

CSS est un langage de style utilisé pour définir l'apparence et la mise en page des pages web.

Javascript

JavaScript est un langage de programmation utilisé pour rendre les pages web interactives et dynamiques.

PHP

PHP est un langage de script côté serveur utilisé pour créer des pages web dynamiques et interagir avec des bases de données.

React

React.js est une bibliothèque JavaScript utilisée pour construire des interfaces utilisateur interactives et réactives.

Node JS

Node.js est un environnement d'exécution JavaScript côté serveur conçu pour créer des applications rapides et évolutives.

My SQL

MySQL est un système de gestion de bases de données relationnelles utilisé pour stocker, organiser et manipuler des données.

Canva

Canva est une plateforme en ligne intuitive utilisée pour créer des designs graphiques et des visuels professionnels.

Webflo

Webflow est une plateforme de conception web qui permet de créer des sites sans coder, en combinant design visuel et développement.
